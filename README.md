# Buurtstem, webapplicatie

Op Buurtstem krijgen de buurtbewoners van de wijk Heihoek de ruimte om inspraak te geven en mee te participeren in het baanbrekende project van de Stad Mechelen en het Agentschap Wegen en verkeer. Zo krijgt elke buurtbewoner de kans om een idee, bekommernis of voorstel achter te laten omtrent de vernieuwing van de Mechelse Vesten Brusselpoort en Ragheno.
